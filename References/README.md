# Basic-project-IA
Esta carpeta contiene el material bibliográfico y fuentes utilizadas en el desarrollo del proyecto.

## Contenido

- Libros y capítulos relevantes
- Documentación técnica
- Enlaces a recursos externos

## Propósito

Centralizar toda la información teórica que respalda el proyecto, facilitando la trazabilidad, comprensión y validación de los métodos utilizados.

## Notas

- Se recomienda incluir referencias en formato estándar (APA, IEEE, etc.)
- Para archivos pesados, considerar enlazar fuentes externas en lugar de almacenarlas directamente
- Mantener actualizada la bibliografía conforme evoluciona el proyecto
